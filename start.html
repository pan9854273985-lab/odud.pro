Установка и запуск курса

Два шага: сначала установить Claude Code, потом скачать файлы курса. Вся установка занимает около 20 минут.

Модуль 0.1 ⏱ ~15 минут

Установка Claude Code

Что нужно перед началом

Требование Подробности
Подписка Claude Pro или Max Claude Pro (от $20/мес) или Claude Max. Бесплатный тариф не поддерживается. Зарегистрироваться: claude.ai
Node.js 18 или выше Бесплатно. Скачать: nodejs.org
Операционная система macOS, Windows 10+ или Linux

Как открыть терминал

Все команды ниже нужно вводить в терминал (командную строку).

Mac:

  1. Нажми Cmd + Space, чтобы открыть Spotlight
  2. Введи «Терминал» и нажми Enter

Windows:

  1. Нажми клавишу Windows
  2. Введи «PowerShell» и нажми Enter

Ты увидишь окно с мигающим курсором — здесь будешь вводить команды.

Шаг 1. Проверь версию Node.js

Введи в терминале и нажми Enter:

node --version

Если видишь v18.0.0 или выше — переходи к шагу 3. Если команда не найдена — переходи к шагу 2.

Шаг 2. Установи Node.js (если нужно)

Рекомендуется: скачай LTS-версию с nodejs.org и запусти установщик.

Альтернативно — через менеджер пакетов:

ПлатформаКоманда
macOS brew install node
Windows choco install nodejs
Важно: после установки Node.js полностью закрой терминал и открой новый. Это нужно, чтобы система подхватила новую версию.

Шаг 3. Установи Claude Code

Mac / Linux 
curl -fsSL https://claude.ai/install.sh | bash
Windows (PowerShell) 
irm https://claude.ai/install.ps1 | iex

Шаг 4. Проверь установку

claude --version

Если видишь номер версии — всё готово. Если command not found — закрой терминал, открой новый и попробуй снова.

Шаг 5. Войди в аккаунт

claude

При первом запуске Claude Code спросит, как войти:

  1. Выбери «Учётная запись Claude с подпиской» (не Anthropic Console)
  2. Войди через браузер
  3. Выбери стиль отображения текста в терминале

После входа увидишь приветственный экран Claude Code.

Шаг 6. Проверь, что всё работает

Внутри Claude Code введи:

/doctor

Команда проверит установку, подписку и конфигурацию. Всё должно быть отмечено зелёными галочками.

Устранение проблем

ОшибкаЧто делать
command not found: claude Закрой терминал полностью, открой новый. Если не помогает — добавь в ~/.zshrc или ~/.bashrc: export PATH=$HOME/.local/bin:$PATH
«Требуется подписка» Убедись, что подписка Claude Pro/Max активна на claude.ai. Попробуй выйти и войти снова.
Версия Node.js ниже 18 Обнови Node.js: brew upgrade node на Mac или скачай новую версию с nodejs.org
«Доступ запрещён» на Mac Попробуй sudo curl -fsSL https://claude.ai/install.sh | bash
Ошибка на Windows Открой PowerShell от имени администратора: правая кнопка мыши → «Запустить от имени администратора»

Модуль 0.2 ⏱ ~5 минут

Скачать курс и начать

Теперь скачаем файлы курса и сразу запустим первый модуль.

Шаг 1. Распакуй файлы курса

После оплаты ты получил доступ к Telegram-каналу. Там закреплён файл hireflow-course.zip — скачай его.

Распакуй в удобную папку. Рекомендуем «Документы»:

Mac / Linux — через терминал 
cd ~/Documents && unzip ~/Downloads/hireflow-course.zip -d hireflow-course
Или вручную (проще)

Дважды кликни по hireflow-course.zip — архив распакуется сам. Перетащи папку hireflow-course в «Документы».

Шаг 2. Запусти курс

Открой терминал в папке hireflow-course:

Mac
  1. Открой Finder → найди папку hireflow-course
  2. Правая кнопка мыши → «Новый терминал в папке»
Windows
  1. Открой папку hireflow-course в Проводнике
  2. Кликни в адресную строку, введи powershell, нажми Enter

Теперь введи:

claude

Claude Code запустится. Затем введи:

/start-1-1

Курс начнётся автоматически.

Частые вопросы

Где найти файл курса?
Файл hireflow-course.zip закреплён в Telegram-канале, доступ к которому ты получил после оплаты. Если не можешь найти — напиши в поддержку: t.me/productodud
Можно сохранить файлы в другую папку?
Да, в любую. Главное — потом запускать claude именно из папки hireflow-course, иначе курс не найдёт свои файлы.
«Новый терминал в папке» не появляется на Mac
Включи в Finder: Настройки → Службы → «Новый терминал в папке». Или просто открой Терминал и введи: cd ~/Documents/hireflow-course
Claude Code запустился, но /start-1-1 не работает
Убедись, что открыл терминал именно в папке hireflow-course — не в родительской папке. Команда ls должна показывать файлы курса.
