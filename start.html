Установка и запуск курса

Выбери свою операционную систему — покажем точные шаги.

Шаг 1 ⏱ ~15 минут

Установка Claude Code

Что нужно

ТребованиеПодробности
Подписка Claude Pro или Max От $20/мес. Зарегистрироваться: claude.ai. Бесплатный тариф не поддерживается.
Node.js 18+ Бесплатно. Скачать: nodejs.org
macOS Любая версия macOS 11+

Как открыть Терминал

  1. Нажми Cmd + Space — откроется Spotlight
  2. Введи «Терминал» и нажми Enter

Ты увидишь окно с мигающим курсором — сюда вводишь команды.

Шаг 1. Проверь Node.js

node --version

Если видишь v18.0.0 или выше — переходи к шагу 3. Если команда не найдена — переходи к шагу 2.

Шаг 2. Установи Node.js (если нужно)

Скачай LTS-версию с nodejs.org и запусти установщик. Или через Homebrew:

brew install node
Важно: после установки закрой Терминал и открой новый.

Шаг 3. Установи Claude Code

curl -fsSL https://claude.ai/install.sh | bash

Дождись сообщения Installation complete!.

Шаг 4. Проверь установку

claude --version

Если видишь номер версии — готово. Если command not found — закрой Терминал, открой новый и попробуй снова.

Шаг 5. Войди в аккаунт

claude
  1. Выбери «Учётная запись Claude с подпиской» (не Anthropic Console)
  2. Войди через браузер
  3. Выбери стиль отображения текста

Шаг 6. Проверь что всё работает

Внутри Claude Code введи:

/doctor

Всё должно быть отмечено зелёными галочками.

Устранение проблем

ОшибкаЧто делать
command not found: claude Закрой Терминал, открой новый. Если не помогает — добавь в ~/.zshrc: export PATH=$HOME/.local/bin:$PATH, потом source ~/.zshrc
«Требуется подписка» Убедись что подписка активна на claude.ai. Попробуй выйти и войти снова.
«Доступ запрещён» Попробуй: sudo curl -fsSL https://claude.ai/install.sh | bash
«Новый терминал в папке» не появляется в Finder Finder → Настройки → Службы → включи «Новый терминал в папке»
Шаг 2 ⏱ ~5 минут

Скачать курс и начать

1. Распакуй файлы курса

В Telegram-канале курса закреплён файл hireflow-course.zip — скачай его.

Дважды кликни по архиву — он распакуется автоматически. Перемести папку hireflow-course в «Документы».

Или через Терминал:

cd ~/Documents && unzip ~/Downloads/hireflow-course.zip -d hireflow-course

2. Открой терминал в папке курса

  1. Открой Finder → найди папку hireflow-course
  2. Правая кнопка мыши → «Новый терминал в папке»

3. Запусти курс

claude

После запуска введи:

/start-1-1

Курс начнётся автоматически.

Частые вопросы

Где найти файл курса?
Файл закреплён в Telegram-канале после оплаты. Если не можешь найти — напиши: t.me/productodud
Можно сохранить курс в другую папку?
Да. Главное — запускать claude именно из папки hireflow-course.
/start-1-1 не работает
Убедись что открыл терминал именно в папке hireflow-course. Команда ls должна показывать файлы курса.

⚠️ Важно для Windows: Claude Code на Windows требует Git Bash. Все команды ниже нужно вводить в Git Bash, а не в PowerShell или CMD — иначе установка упадёт с ошибками.
Шаг 1 ⏱ ~20 минут

Установка Claude Code

Что нужно

ТребованиеПодробности
Подписка Claude Pro или Max От $20/мес. Зарегистрироваться: claude.ai. Бесплатный тариф не поддерживается.
Node.js 18+ Бесплатно. Скачать: nodejs.org
Git Bash — обязательно Скачать: git-scm.com/downloads/win → установи с настройками по умолчанию (Next → Next → Install)
Windows 10+ 64-bit

Шаг 1. Установи Git Bash

Скачай с git-scm.com/downloads/win и запусти установщик. Все настройки оставь по умолчанию — просто жми Next → Install.

После установки найди Git Bash в меню Пуск и открой его. Все следующие шаги — только в Git Bash.

Шаг 2. Проверь Node.js

В Git Bash введи:

node --version

Если видишь v18.0.0 или выше — переходи к шагу 4. Если команда не найдена — переходи к шагу 3.

Шаг 3. Установи Node.js (если нужно)

Скачай LTS-версию с nodejs.org и запусти установщик.

Важно: после установки полностью закрой Git Bash и открой снова.

Шаг 4. Установи Claude Code

В Git Bash введи:

curl -fsSL https://claude.ai/install.sh | bash

Дождись сообщения Installation complete!.

Не используй PowerShell для этой команды — только Git Bash. PowerShell выдаст ошибки синтаксиса.

Шаг 5. Проверь установку

claude --version

Если видишь номер версии — готово. Если command not found — смотри раздел «Устранение проблем» ниже.

Шаг 6. Войди в аккаунт

claude
  1. Выбери «Учётная запись Claude с подпиской» (не Anthropic Console)
  2. Войди через браузер
  3. Выбери стиль отображения текста

Шаг 7. Проверь что всё работает

Внутри Claude Code введи:

/doctor

Всё должно быть отмечено зелёными галочками.

Устранение проблем

ОшибкаЧто делать
Красные ошибки при установке (синтаксис, «лексема») Ты запускаешь команду в PowerShell. Закрой его и открой Git Bash — повтори установку там.
requires git-bash Установи Git Bash: git-scm.com/downloads/win → Next → Install. Закрой терминал, открой Git Bash.
claude: command not found в Git Bash Путь не добавился автоматически. Добавь вручную: открой Пуск → «Переменные среды» → «Переменные среды для текущего пользователя» → Path → «Изменить» → «Создать» → вставь C:\Users\ТВО_ИМЯ\.local\bin (замени ТВО_ИМЯ на своё имя пользователя). ОК → закрой Git Bash → открой снова.
«Требуется подписка» Убедись что подписка активна на claude.ai. Попробуй выйти и войти снова.
Ничего не помогает Напиши в поддержку со скриншотом ошибки: t.me/productodud
